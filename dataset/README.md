# Dataset

Project ini otomatis membaca dataset dari `dataset/` bila folder tersebut ada.

Pada workspace saat ini, data asli sudah tersedia di:

- `dataset_bisindo/train`
- `dataset_bisindo/val`

Jadi Anda tidak perlu memindahkan data. Jika nanti ingin mengikuti struktur generik pada dokumentasi, Anda bisa mengganti nama `dataset_bisindo` menjadi `dataset` atau mengubah argumen `--train-dir` dan `--val-dir` saat training.
